To run the code:  

1. Goto `server` -> `index.js` -> `line 19` and change the credentials according to your MongoDB connection  
2. `cd Memories-Keeper`  
3. Open two consoles:  

On console 1-  
`cd client`  
`npm install`  
`npm start`  

On console 2-  
`cd server`  
`npm install`  
`npm start`
